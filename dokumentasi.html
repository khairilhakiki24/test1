Manual Book AuraPrompt

Panduan lengkap Instalasi, Penggunaan, dan Modifikasi (White-label) Web App PWA Anda.

1. Panduan Fitur Utama

1

Manajemen Prompt & Anti-Data Loss

Aplikasi ini menyimpan data murni di browser pengguna menggunakan IndexedDB. Anda tidak perlu menyewa database (MySQL/Firebase). Selain itu, terdapat fitur State Persistence: Jika Anda sedang mengetik prompt panjang lalu tidak sengaja keluar/refresh web, tulisan Anda akan tersimpan sebagai Draft.

2

Backup & Restore (Excel)

Buka menu Pengaturan. Anda bisa men-download seluruh prompt Anda menjadi file .xlsx (Excel). File ini bisa dipindahkan ke HP/Laptop lain, lalu di-import kembali melalui tombol "Import dari Excel".

3

PWA Offline First

Setelah website dibuka pertama kali, matikan koneksi internet (Airplane Mode). Refresh halaman web. Aplikasi akan TETAP BERJALAN 100% karena menggunakan teknologi Service Worker. Tidak akan ada layar "Dinosaurus Chrome".

2. Panduan Modifikasi & Branding (White Label)

Anda dapat menjual ulang aplikasi ini dengan nama dan warna lain (SaaS / White Label).

  • Ganti Nama Aplikasi: Buka aplikasi web, masuk ke tab Pengaturan, ketik nama baru (Misal: "PromptMaster"), lalu klik Simpan.
  • Ganti Warna Tema: Di menu Pengaturan yang sama, klik Color Picker dan pilih warna brand Anda (misal Merah atau Hijau). Aplikasi akan seketika berubah warna tanpa perlu ngoding.
  • PENTING: Ganti Icon PWA
    Buat folder bernama assets di sebelah file index.html. Masukkan 2 file gambar berformat .png (jangan JPG/SVG):
    • assets/icon-192x192.png (Ukuran wajib 192x192 pixel)
    • assets/icon-512x512.png (Ukuran wajib 512x512 pixel)
    Ini sangat penting agar saat aplikasi di "Add to Home Screen" di HP, logonya tidak pecah.

3. Panduan Hosting / Instalasi

Karena ini adalah Single File Application berbasis Statis, hostingnya sangat mudah dan bisa GRATIS 100%.

Struktur Folder yang harus diupload:

  • index.html
  • manifest.json
  • sw.js
  • assets/ (folder berisi icon PNG Anda)

Opsi 1: Menggunakan cPanel (Shared Hosting)

  1. Login ke cPanel hosting Anda.
  2. Buka File Manager -> Masuk ke folder public_html atau folder subdomain Anda.
  3. Klik tombol Upload, pilih file-file di atas. Selesai!
  4. Akses website Anda (wajib HTTPS:// agar PWA bisa diinstal).

Opsi 2: Gratis via GitHub Pages / Vercel

  1. Buat akun GitHub, buat repository baru (public).
  2. Upload file-file ke repository tersebut.
  3. Pergi ke Settings -> Pages -> Pilih Branch "main", lalu Save.
  4. Tunggu 1-2 menit, web Anda sudah live dengan URL gratis dari GitHub dan otomatis HTTPS.
